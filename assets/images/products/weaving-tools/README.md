# Weaving Tools Products

This folder contains all Weaving Tools product images.

## Subcategories

- `bodkins/` - Bodkins subcategory
- `[other-subcategories]/` - Other weaving tool subcategories

## Structure

Each subcategory should have:
- `subcategory-preview.jpg` - Preview image for category page
- `[product-name]/` - Individual product folders

## Example

```
weaving-tools/
├── bodkins/
│   ├── subcategory-preview.jpg
│   ├── walnut-steel-bodkin/
│   │   ├── 01-main.jpg
│   │   ├── 02-detail.jpg
│   │   └── 03-side-view.jpg
│   └── cherry-bodkin/
│       └── [product images]
└── [other-subcategories]/
    └── [same structure]
```
