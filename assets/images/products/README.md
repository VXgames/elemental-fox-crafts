# Products Folder

This folder contains all product images organized by category → subcategory → product.

## Structure

```
products/
├── [category-name]/              (e.g., copper-works)
│   ├── [subcategory-name]/       (e.g., copper-spoons)
│   │   ├── subcategory-preview.jpg
│   │   └── [product-name]/
│   │       ├── 01-main.jpg
│   │       ├── 02-detail.jpg
│   │       └── 03-side-view.jpg
│   └── [other-subcategories]/
└── [other-categories]/
```

## Categories

- `copper-works/` - Copper Works products
- `weaving-tools/` - Weaving Tools products
- `wands/` - Wands products
- `cat-toys/` - Cat Toys products
- `knives/` - Knives products

## Image Specifications

### Subcategory Preview
- **Size**: 800×800px (square)
- **File Size**: 100-150 KB

### Product Images
- **Size**: 800×800px (square) for listings
- **Size**: 1200×1200px (square) for detail pages
- **Format**: JPEG
- **Quality**: 80-85%
- **File Size**: 100-150 KB (800×800) or 200-300 KB (1200×1200)

## Naming Convention

- **Subcategory preview**: `subcategory-preview.jpg`
- **Product images**: `01-main.jpg`, `02-detail.jpg`, `03-side-view.jpg`, etc.
- Use two-digit numbering (01, 02, 03) for proper sorting

## Usage

- Subcategory previews are used on category pages
- Product images are used on subcategory listing pages and product detail pages
