# Cat Toys Products

This folder contains all Cat Toy product images.

## Subcategories

Create subcategory folders as needed:
- `[subcategory-name]/` - Cat toy subcategories

## Structure

Each subcategory should have:
- `subcategory-preview.jpg` - Preview image for category page
- `[product-name]/` - Individual product folders

## Example

```
cat-toys/
├── [subcategory-name]/
│   ├── subcategory-preview.jpg
│   ├── [product-name]/
│   │   ├── 01-main.jpg
│   │   ├── 02-detail.jpg
│   │   └── 03-side-view.jpg
│   └── [other-products]/
└── [other-subcategories]/
    └── [same structure]
```

## Product Image Structure

Each product folder should contain:
- `01-main.jpg` - Main product image (800×800px)
- `02-detail.jpg` - Detail view (800×800px)
- `03-side-view.jpg` - Side view (800×800px)
- Additional images as needed (04, 05, etc.)

## Image Specifications

- **Size**: 800×800px (square) for listings
- **Size**: 1200×1200px (square) for detail pages
- **Format**: JPEG
- **Quality**: 80-85%
- **File Size**: 100-150 KB (800×800) or 200-300 KB (1200×1200)
