# Copper Spoons

This folder contains all Copper Spoons product images.

## Files

- `subcategory-preview.jpg` - Preview image for the Copper Works category page

## Product Folders

Create a folder for each product:
- `copper-tea-spoon/` - Copper Tea Spoon
- `copper-coffee-scoop/` - Copper Coffee Scoop
- `[other-products]/` - Other copper spoon products

## Product Image Structure

Each product folder should contain:
- `01-main.jpg` - Main product image (800×800px)
- `02-detail.jpg` - Detail view (800×800px)
- `03-side-view.jpg` - Side view (800×800px)
- Additional images as needed (04, 05, etc.)

## Image Specifications

- **Size**: 800×800px (square) for listings
- **Size**: 1200×1200px (square) for detail pages
- **Format**: JPEG
- **Quality**: 80-85%
- **File Size**: 100-150 KB (800×800) or 200-300 KB (1200×1200)
