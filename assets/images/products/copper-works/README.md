# Copper Works Products

This folder contains all Copper Works product images.

## Subcategories

- `copper-spoons/` - Copper Spoons subcategory
- `copper-ladles/` - Copper Ladles subcategory
- `copper-jewellery/` - Copper Jewellery subcategory

## Structure

Each subcategory should have:
- `subcategory-preview.jpg` - Preview image for category page
- `[product-name]/` - Individual product folders

## Example

```
copper-works/
├── copper-spoons/
│   ├── subcategory-preview.jpg
│   ├── copper-tea-spoon/
│   │   ├── 01-main.jpg
│   │   ├── 02-detail.jpg
│   │   └── 03-side-view.jpg
│   └── copper-coffee-scoop/
│       └── [product images]
└── copper-ladles/
    └── [same structure]
```
