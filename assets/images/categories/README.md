# Categories Folder

This folder contains hero/header images for each category page.

## Structure

One image per category:
- `copper-works.jpg` - Category hero for Copper Works
- `weaving-tools.jpg` - Category hero for Weaving Tools
- `wands.jpg` - Category hero for Wands
- `cat-toys.jpg` - Category hero for Cat Toys
- `knives.jpg` - Category hero for Knives

## Naming Convention

Use kebab-case matching the category name:
- `copper-works.jpg`
- `weaving-tools.jpg`
- `wands.jpg`
