# Featured Items Folder

This folder contains the 3 best-selling items that appear in the Shop mega-menu dropdown.

## Structure

Place one image per featured item:
- `item-01.jpg` - First featured item (600×600px)
- `item-02.jpg` - Second featured item (600×600px)
- `item-03.jpg` - Third featured item (600×600px)

## Naming Convention

Use descriptive names:
- `walnut-steel-bodkin.jpg`
- `cherry-wood-wand.jpg`
- `natural-cat-wand.jpg`

## Image Specifications

- **Size**: 600×600px (square)
- **Format**: JPEG
- **Quality**: 80-85%
- **File Size**: 80-120 KB

## Usage

These images are displayed in the mega-menu on the Shop dropdown.
Update the HTML in `index.html` and other pages to reference these images.
